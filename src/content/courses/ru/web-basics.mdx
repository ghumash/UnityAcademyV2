# Введение

В этом курсе вы изучите основы верстки и программирования для веба: HTML, CSS и JavaScript.

## Цели курса

- Понять как устроены веб-страницы
- Научиться базовой вёрстке
- Освоить основы JavaScript

## Требования

- Компьютер с современным браузером
- Готовность учиться 5–7 часов в неделю

## Программа

### Модуль 1: HTML

- Семантика, теги, атрибуты
- Ссылки, изображения, списки, таблицы

### Модуль 2: CSS

- Селекторы, каскад, специфичность
- Flexbox и Grid
- Адаптивная вёрстка

### Модуль 3: JavaScript

- Переменные, типы, функции
- DOM и события
- Модули и сборщики

## Результаты

К концу курса вы сверстаете адаптивную страницу и добавите интерактивность на JS.

> Совет: начните с маленьких проектов и постепенно усложняйте.
