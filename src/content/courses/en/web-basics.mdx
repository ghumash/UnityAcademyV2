# Introduction

This course introduces the fundamentals of web development: HTML, CSS, and JavaScript.

## Goals

- Understand how web pages are structured
- Learn the basics of layout
- Get started with JavaScript

## Requirements

- A computer with a modern browser
- Willingness to learn 5–7 hours per week

## Curriculum

### Module 1: HTML

- Semantics, tags, attributes
- Links, images, lists, tables

### Module 2: CSS

- Selectors, cascade, specificity
- Flexbox and Grid
- Responsive layout

### Module 3: JavaScript

- Variables, types, functions
- DOM and events
- Modules and bundlers

## Outcomes

By the end, you will build a responsive page and add interactivity with JS.

> Tip: Start with small projects and increase complexity gradually.
