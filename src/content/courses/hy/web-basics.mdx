# Ներածություն

Այս կուրսում Դուք կծանոթանաք վեբ ծրագրավորման հիմունքներին՝ HTML, CSS և JavaScript։

## Նպատակներ

- Հասկանալ, թե ինչպես են կառուցված վեբ էջերը
- Սովորել դասավորության հիմունքները
- Սկսել JavaScript-ից

## Պահանջներ

- Համակարգիչ՝ ժամանակակից բրաուզերիով
- Պատրաստակամություն՝ շաբաթական 5–7 ժամ սովորելու

## Ծրագիր

### Մոդուլ 1․ HTML

- Սեմանտիկա, թեգեր, ատրիբուտներ
- Հղումներ, պատկերներ, ցուցակներ, աղյուսակներ

### Մոդուլ 2․ CSS

- Սելեկտորներ, կասկադ, սպեցիֆիկություն
- Flexbox և Grid
- Հարմարվողական դասավորություն

### Մոդուլ 3․ JavaScript

- Փոփոխականներ, տիպեր, ֆունկցիաներ
- DOM և իրադարձություններ
- Մոդուլներ և հավաքիչներ

## Արդյունքներ

Կուրսի ավարտին դուք կստեղծեք հարմարվողական էջ և կավելացնեք ինտերակտիվություն JS-ով։

> Խորհուրդ․ սկսեք փոքր նախագծերից և աստիճանաբար բարդացրեք։
